# 🌦️ Weather App

Bu React yordamida yaratilgan zamonaviy Weather App loyihasi.
Foydalanuvchilar istalgan shahar ob-havosini qidirib, real vaqt ma’lumotlarini ko‘rishlari mumkin.

## 🌐 Live Demo

🔗 [https://your-weather-app-link.com](https://alibekshomurotov.github.io/Weather-App/)

---

## 📌 Loyiha Haqida

Ushbu loyiha foydalanuvchilarga:

* Shahar bo‘yicha ob-havo qidirish
* Haroratni ko‘rish
* Namlik va shamol tezligini bilish
* Zamonaviy interfeys orqali qulay foydalanish imkonini beradi.
